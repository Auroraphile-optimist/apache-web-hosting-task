Default document root & simple index

Default docroot: /var/www/html on RHEL.

Create a simple index file (replace existing or create new):

sudo mv /var/www/html/index.html /var/www/html/index.html.orig 2>/dev/null || true
cat <<'HTML' | sudo tee /var/www/html/index.html
<!doctype html>
<html>
<head><meta charset="utf-8"><title>Default Apache Page</title></head>
<body>
  <h1>It works â€” Default site</h1>
  <p>Served from /var/www/html</p>
</body>
</html>
HTML


Set permissions:

sudo chown -R root:root /var/www/html
sudo chmod -R 755 /var/www/html


Test:

curl http://localhost/
